# Eco Lifestyle Agent 🌍

## Overview

Eco Lifestyle Agent is an AI-powered sustainability assistant built using IBM watsonx Orchestrate.

The agent helps users adopt environmentally friendly habits through personalized recommendations, recycling guidance, sustainable travel suggestions, eco-friendly product recommendations, and government sustainability schemes.

---

## Problem Statement

Problem Statement No. 6 – Eco Lifestyle Agent

The challenge is to empower users to adopt a greener lifestyle through personalized, practical sustainability guidance using Retrieval-Augmented Generation (RAG).
